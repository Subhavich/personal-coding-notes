Input & Change

This paragraph can be changed if input is filled

- Input event is detected as data in forms is being changed

- Change event is detected as form is modified and exited

===Input===

input.addEventListener("input", (e) => { 

para.innerHTML = `You pressed "${e.data}"`;

 });

===Change===

input.addEventListener("change", (e) => { 

para.innerHTML = `Current Value is "${input.value}"`;

 });
